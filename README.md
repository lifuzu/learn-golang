

### Preparation
```
$ export GOPATH=$HOME/Workspace/Go
$ export PATH=$PATH:$GOPATH/bin
$ go get github.com/lifuzu/learn-golang
```

### Run main to learn golang
```
$ cd $GOPATH/src/github.com/lifuzu/learn-golang
$ go run main.go z.go
```

### Run web server, supported by Echo
```
$ cd $GOPATH/src/github.com/lifuzu/learn-golang
$ go run server.go
```
